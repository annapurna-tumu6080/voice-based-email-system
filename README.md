# voice-based-email-system
Developed a Voice-Based Email System that allows users to send and read emails using voice commands. It uses speech-to-text for composing messages and text-to-speech for reading emails, enabling hands-free operation and improving accessibility for visually impaired users.

It integrates speech recognition, natural language processing, and email protocols (SMTP/IMAP) to perform email operations securely and efficiently. The application enhances user experience by eliminating the need for keyboard interaction and providing an intuitive voice-driven interface.
